# AI-Based Water Demand Forecasting & Pump Optimization

This project uses Machine Learning to forecast future water demand and simulate optimized pumping schedules to reduce energy consumption and over-pumping in institutional environments.

---

## Live Application
https://water-demand-forecasting-app-mqhkzgifiza7lc7mckcpvf.streamlit.app/

---

## Project Features
- Water demand forecasting using Machine Learning
- Interactive Streamlit dashboard
- Pump optimization simulation
- Energy savings estimation
- Carbon emission reduction analysis
- Country-wise water consumption visualization

---

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Plotly
- Pandas
- Machine Learning (Linear Regression)

---

## Project Objective
To develop an AI-based forecasting system that predicts future water demand and demonstrates how optimized pumping strategies can reduce electricity consumption and carbon emissions.

---

## Sustainability Impact
This system demonstrates how predictive analytics can support **sustainable water resource management** and contribute to:

- **SDG 6** – Clean Water and Sanitation  
- **SDG 13** – Climate Action
